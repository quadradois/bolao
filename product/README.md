# Product Book

O Product Book é a fonte oficial do conhecimento do produto.

## Objetivo
Documentar regras de negócio, arquitetura funcional e especificações necessárias para que a plataforma seja desenvolvida de forma consistente.

## Estrutura
- FOUNDATION
- COMPETITION
- PARTICIPANT
- FINANCIAL
- COMMUNICATION
- ADMINISTRATION
- PLATFORM
- AI
- SPECS

## Regra de Ouro
Nenhuma funcionalidade será implementada antes de existir uma documentação correspondente neste Product Book.
