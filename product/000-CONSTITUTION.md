# Constituição da Plataforma

## Objetivo
Este documento estabelece os princípios permanentes da plataforma.

## Artigos
1. Transparência acima de tudo.
2. O Competition Engine é a única autoridade para evolução de uma competição.
3. Após o fechamento das inscrições, participantes, dezenas e pagamentos tornam-se imutáveis.
4. Todo Bolão encerrado torna-se patrimônio histórico da plataforma.
5. Toda regra variável deve ser configurável, nunca fixa no código.
6. O administrador configura; a plataforma executa.
7. Toda mudança importante gera um evento de domínio.
8. O GitHub é a fonte oficial da verdade do produto.
9. O participante acompanha uma competição, não opera um sistema.
10. Toda documentação deve ser escrita em formato AI First para humanos e agentes de IA.
